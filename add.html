How to submit new entries

Want to submit a new public company?

  1. MUST be publicly traded, please include symbol and exchange (ie NASDAQ:SQ)
  2. MUST include filing or announcement link
  3. MUST include total BTC held in Treasuries
  4. MUST include purchase price and/or total in the balance sheet as of last filing in USD
  5. IF you have ALL those items create an issue here

Want to submit a new AUM/NAV/ETF/ETC type company?

  1. MUST be publicly traded or "OTCX" widely available, please include symbol and exchange (ie OTCX:GBTC)
  2. MUST include last financial report
  3. MUST include total BTC
  4. MUST include purchase price and/or total in the balance sheet as of last filing in USD
  5. IF you have ALL those items create an issue here

Want to submit a new Large Private Company?

  1. MUST include legal name and ideally a profile on financial site (ie Bloomberg)
  2. MUST have some notoriety so that is easier to validate the data. (ie Stone Ridge @ NYDIG)
  3. MUST include a link to financial report or some other form of verification
  4. MUST include total BTC held, minimun $50M of bitcoin to qualify
  5. MUST include purchase price of each position or total basis price as of last report in USD
  6. MUST include company valuation and/or total for assets under management in USD
  7. IF you have ALL those items create an issue here
